# 小工具常用操作右侧 Tab 设计

## 目标

将小工具中的“常用操作”从块状卡片组合改为右侧竖向 Tab 导航，左侧展示当前选中操作的内容面板。

## 范围

- 保留小工具页面外层主 Tab 结构不变。
- 只调整 `CommonActions.vue` 内部布局。
- 当前纳入两个操作页签：
  - `命令托管`
  - `端口进程管理`

## 设计

- 顶部保留“常用操作”的标题和说明。
- 主内容区改为单个 `el-tabs`，使用 `tab-position="right"`。
- Tab 标签作为右侧菜单，内容区承载当前操作面板。
- 每个操作继续复用原有表单、按钮和结果区逻辑，不改后端接口语义。
- 移除原有“后续可扩展”占位卡，后续新增功能直接扩展为新的 Tab。

## 交互约束

- 默认激活 `命令托管`。
- 切换 Tab 不重置页面级状态。
- 移动端降级为上下布局，但仍保留 Tab 语义。

## 验证

- “常用操作”能正常进入。
- 右侧显示竖向 Tab 菜单。
- `命令托管` 与 `端口进程管理` 可正常切换。
- 原有操作能力不回归。
