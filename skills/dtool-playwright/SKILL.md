# dtool-playwright

## 功能索引

- 通过 smart-link 登录能力打开目标页面
- MCP 模式接管已登录浏览器会话
- Playwright 持久化目录模式接管浏览器
- 登录后抓取页面首个接口请求头
- 网页截图

详细说明见 [references/detail.md](references/detail.md)
