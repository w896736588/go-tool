# dtool-git

## 功能索引

- 上传本地文件到远程项目目录
- 查询当前分支、拉取代码、切换分支
- 查看当前分支相对基线的改动文件列表
- 查看单文件 diff
- 查看前端/后端常见文件类型的完整改动及 diff

详细说明见 [references/detail.md](references/detail.md)
